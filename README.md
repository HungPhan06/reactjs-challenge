# Setup & Installation
1. Install dependencies:
npm install
or
yarn install

2. Configure environment variables:
 - VITE_REACT_APP_BASE_URL=your_endpoint

Example: 
 - VITE_REACT_APP_BASE_URL=https://nodejs-challenge-production.up.railway.app

3. Run the development server
npm run dev
or
yarn dev
